# Trading Model Training System v4.1.1 Premium Visual Terminal

Vite + React + Tailwind 交易模型训练系统。当前包已清理 `node_modules` 与 `dist`，可直接上传 GitHub / Vercel。

## 本地运行

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
npm run preview
```

## 部署说明

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

## 注意

不要上传 `node_modules` 和 `dist`。依赖通过 `npm install` 自动生成，`dist` 通过 `npm run build` 自动生成。
